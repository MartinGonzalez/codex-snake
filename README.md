# Web game template (Pages + PR previews)

This branch is intended as a starting point for small web games.

## What you get
- GitHub Pages deploy on every push to `main`
- PR preview deploys at `/pr/<PR_NUMBER>/`
- Automatic cleanup when PR closes

## Where it is implemented
- `.github/workflows/pages.yml`

## How to use
- Copy this repo into a new one (or start a new game by replacing `index.html` and adding your files)
- Open PRs: previews publish automatically
